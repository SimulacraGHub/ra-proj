Nx React Monorepo Portfolio Project
Overview

This repository contains a portfolio demonstration project built using Nx monorepo architecture and React.

It showcases my ability to structure scalable front-end applications, implement routing, create reusable shared libraries, and manage a modular codebase using modern TypeScript tooling.

This project forms part of my professional portfolio as a junior full-stack developer.

    Tech Stack

React (Functional Components + Hooks)

TypeScript

Nx Monorepo Architecture

React Router

Shared Component Library

Node.js + npm

    Architecture Highlights

Monorepo structure using Nx

Separation of concerns between application and shared libraries

Centralized configuration via tsconfig.base.json

Reusable UI components housed in libs/shared

Clean routing setup with React Router

Scalable project structure for future expansion

    Running the Project Locally

From the project root:

npm install
npx nx serve react-app

The app will run at:

http://localhost:4200
Project Structure
react-app/ → Main React application
libs/shared/ → Reusable shared components
nx.json → Nx workspace configuration
